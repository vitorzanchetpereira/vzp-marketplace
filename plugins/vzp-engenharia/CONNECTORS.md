# Conectores

Este plugin descreve fluxos que usam ferramentas externas. Cada pessoa da equipe
deve conectar as ferramentas abaixo **com a própria conta** (o plugin não guarda
login nem credencial de ninguém).

| Categoria            | Ferramenta usada no escritório | Para quê                                   |
| -------------------- | ------------------------------ | ------------------------------------------ |
| Gestão de tarefas    | Asana (workspace BASE Empreendimentos) | Contratos, NF, recibos, medições, compras |
| E-mail / Calendário  | Outlook ou Gmail               | Ler e-mails, organizar tarefas             |
| Arquivos / Nuvem     | SharePoint (Base Empreendimentos - Documentos) | Pastas das obras, contratos, fiscal |
| Navegador            | Claude no Chrome               | Anexar PDFs em tasks do Asana              |

## Como conectar

Nas configurações de conectores do Cowork/Claude, cada usuário autoriza Asana,
e-mail e a nuvem com a conta dele. Depois de conectado, as skills deste plugin
passam a funcionar automaticamente.
