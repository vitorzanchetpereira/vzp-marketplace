# Arquivamento por obra — Base Empreendimentos

Estrutura por obra (ex.: `Ob em Andam/Ob-UG-BASE-Sorriso-MT`, onde UG = Urbanit Garden):

- **NF-e recebidas** → `Fisc/01-Nfe-Recebidas/`
- **Recibos** → `Fin/Recibos/`
- **Docs contratuais de fornecedor** → `Contratos/<Fornecedor>/`

## Padrão de nome de arquivo

`AAMMDD-UG-CATEGORIA-Descricao.pdf`

Exemplo: `260702-UG-FISCAL-NFe80-CriarStudio-1aMedicao.pdf`

- `AAMMDD` = data
- Sigla da obra (ex.: `UG`)
- `CATEGORIA` (ex.: FISCAL, FIN)
- Descrição curta sem espaços
