# Padrão de contratos — VZP / Base Empreendimentos

Padrão definido pela equipe jurídica/incorporação (contato: Rosana — grupo
"INC - CONST - JUR - BASE EMPREENDIMENTOS").

## Qualificação das partes no preâmbulo

Qualificação completa de PJ / administrador, **nesta ordem**:

1. Razão social
2. "pessoa jurídica de direito privado, inscrita sob o CNPJ nº ..."
3. Endereço da sede — **CEP sempre antes da cidade**
   Ex.: "...CEP 78890-148, na cidade de Sorriso, Estado de Mato Grosso"
4. Representante: "neste ato representada pelo administrador não sócio [Nome],
   brasileiro, [estado civil], [profissão], inscrito no CPF sob o nº ...,
   portador do RG nº ... SESP/MT, residente e domiciliado na [endereço
   residencial], CEP ..., na cidade de ..., Estado de ..."

**Regra prática (enfatizada pelo jurídico):** "TOC no CEP, sempre antes da cidade."

## Revisão de contrato

- Ao incorporar parecer jurídico, registrar o **histórico de disposição de cada
  item** (acatado / não acatado e por quê) em documento separado.
- Cláusulas novas inseridas inline exigem **renumeração** de todo o restante.
- Nem todo item do parecer é acatado — confirmar com a CONTRATANTE itens sensíveis
  (ex.: Propriedade Intelectual) antes de alterar.

## Dados das partes

Os dados de cada SPE/empreendimento (CNPJ, endereço, administrador, CPF/RG) ficam
com o setor jurídico. Pergunte ao usuário ou consulte a ficha da obra ao preencher
o preâmbulo — não presuma dados pessoais.

Arquivar contratos conforme a skill **arquivamento-obras**; se registrar no Asana,
o responsável é a Duana (skill **suprimentos-asana**).
